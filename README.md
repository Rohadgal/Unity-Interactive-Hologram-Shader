# Programming and Visual Effects in Video Games: Creating an Interactive Shader in Unity

## Overview

This project accompanies the class **"Programming and Visual Effects in Video Games: Creating an Interactive Shader in Unity."**

Through this hands-on exercise, students will explore how gameplay systems can communicate with **Shader Graph** to create interactive visual effects in real time. The project combines **C# programming**, shaders, materials, and visual feedback systems to enhance the player experience.

---

## Learning Objectives

By the end of this activity, students will be able to:

* Understand the fundamentals of **Shader Graph** in Unity.
* Create **interactive visual effects** using material parameters.
* Connect **C# programming** with visual systems.
* Generate **visual feedback** based on gameplay variables.

---

## Topics Covered

* Shader Graph
* Materials and Shader Parameters
* Component-Based Programming in Unity
* Communication Between Scripts and Shaders
* Visual Feedback for Players
* Basic Gameplay Systems
* Real-Time Variables
* Interactive Visual Effects

## Class Activity

During this session, students will build a hologram-style visual effect using **Shader Graph** and connect it to a gameplay system through **C# scripting**.

Students will observe how gameplay variables can influence shader behavior in real time, allowing developers to create more immersive and communicative player experiences.
